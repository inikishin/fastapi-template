# Шаблон микросервиса на FastAPI

Данный шаблон предназначен для быстрого развертывания микросервиса на [FastAPI](https://fastapi.tiangolo.com/) и включает в себя следующие компоненты:

- данный readme.md файл;
- структуру каталогов микросервиса;
- requirements.txt для установки зависимостей;
- Makefile с различными командами, описание которых дано ниже.

## Makefile описание команд

```shell
make install
```

Установка необходимых зависимостей.

```shell
make run
```

Запуск сервера локально.

```shell
make lint
```

Запуск линтеров.

```shell
make test
```

Запуск тестов.

```shell
make migrate msg=migration-description
```

Создать автомиграцию с описанием из параметра msg

```shell
make upgrade
```

Применить все миграции

```shell
make downgrade
```

Откатить последнюю миграцию

```shell
make history
```

Показать историю миграций алембика

## Архитектура микросервиса

Микросервис представляет собой многослойную архитектуру, которая включает в себя следующие слои:

1. Уровень1: API - методы апи, их описание, схемы, валидации, обработки исключений.
2. Уровень2: Сервисы - включают в себя бизнес логику.
3. Уровень3: Менеджеры моделей - предназначены для работы с БД
4. Уровень3: Внешние сервисы - предназначены для работы с внешними сервисами (сторонний api, emails и т.п)

## Текущий шаблон

Текущий шаблон представляет собой реализацию извлечения данных по облигациям с api Московской фондовой биржи, сохранения их в БД и логику формирования лестницы облигаций на основе передаваемых параметров. Описание функциональности представлено в файле user_story.md.
